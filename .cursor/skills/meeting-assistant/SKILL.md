# WPS365 会议助手

一个全面的 WPS365 日程管理工具集，用于高效管理会议安排、查询参与者可用时间以及批量操作日程。

## 核心功能

### 1. 批量查询用户忙闲状态
- 查询多个用户在指定时间段内的繁忙时间
- 支持自定义时间范围查询
- 返回每个用户的详细忙闲时间段
- 帮助快速找到合适的会议时间

### 2. 创建日程（包含视频会议）
- 创建带有视频会议链接的日程
- 自动配置会议设置（录制、权限、提醒等）
- 支持自定义会议标题和时间
- 返回日程 ID 用于后续操作

### 3. 批量添加日程参与者
- 向已有日程批量添加参与者
- 自动发送通知给所有参与者
- 支持通过用户 ID 添加
- 返回添加结果确认

### 4. 一键创建会议并邀请参与者
- 组合创建日程和添加参与者的流程
- 一步完成会议的创建和人员邀请
- 自动处理错误并返回详细状态
- 最适合快速组织会议

## 功能详解

### 1. 批量查询用户忙闲状态

查询多个用户在指定时间范围内的日程占用情况。

**返回示例：**
```python
[
    "用户1734216126的繁忙时间段有：2026-01-28T10:00至2026-01-28T11:00, 2026-01-28T14:00至2026-01-28T15:30",
    "用户1384381367没有繁忙时间段",
    "用户1610124305的繁忙时间段有：2026-01-28T16:00至2026-01-28T17:00"
]
```

**命令行使用：**
```bash
python wps365_calendar.py get_free_busy_list \
  --start_time "2026-01-28 09:00" \
  --end_time "2026-01-29 18:00" \
  --user_ids "1734216126, 1384381367, 1610124305"
```

### 2. 创建会议日程

创建一个新的会议日程事件，自动包含 WPS 视频会议链接。

**返回示例：**
```
会议日程id（event_id）为: 122606009
```

**会议自动配置：**

- 忙闲状态：busy（繁忙）
- 自动录制：开启（含 AI 纪要）
- 加入权限：任何人可加入
- 入会静音：开启
- 录制查看权限：参与者可查看
- 提前提醒：15 分钟

**命令行使用：**
```bash
python wps365_calendar.py create_calendar_event \
  --start_time "2026-01-29 14:00" \
  --end_time "2026-01-29 15:00" \
  --summary "产品需求评审会议"
```

### 3. 批量添加会议日程参与者

向已创建的日程批量添加参与者。

**返回示例：**
```
用户 1384381367, 1610124305 已添加至日程
```

**命令行使用：**
```bash
python wps365_calendar.py batch_create_attendees \
  --event_id "122606009" \
  --user_ids "1384381367, 1610124305"
```

### 4. 创建会议日程并批量添加参与者

组合操作：一步完成会议创建和参与者邀请。

**返回示例：**
```
已创建日程'每周项目复盘会议'（id: 122606009），并添加参与者：1384381367, 1610124305
```

**命令行使用：**
```bash
python wps365_calendar.py create_event_with_attendees \
  --start_time "2026-01-29 16:00" \
  --end_time "2026-01-29 17:00" \
  --summary "每周项目复盘会议" \
  --user_ids "1384381367, 1610124305"
```
